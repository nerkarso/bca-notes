A collection of notes of the BCA course.

## Development

### Installation

```sh
$ git clone https://github.com/nerkarso/bca-notes.git
$ cd bca-notes
$ yarn install
```

### Commands

```sh
# Compiles and hot-reloads for development
yarn start

# Compiles and minifies for production to build/
yarn build
```
